## Basic LAN With Three PCs

## Objectives
Connect three PCs via Cisco switch to establish a simple Lan and verify connectivity.

## Devices Used
- 3 PCs (PC0, PC1, PC2)
- 1 Cisco 2960 Switch

## IP Configuration
- PC0: 192.168.1.1, 255.255.255.0
- PC1: 192.168.1.2, 255.255.255.0
- PC2: 192.168.1.3, 255.255.255.0

## Steps Performed
1. Added three PCs and one switch in the Cisco Packet Tracer.
2. Connected each PC to the switch using copper straight-through cables.
3. Configured static IP addressing on all the three PCs.
4. Tested connectivity by pinging PC1 and PC2 from PC0.

## Test Results
The ping from PC0 to PC1 and to PC2 was successful, connfirming communication over LAN.

## Skills Demonstrated
- Basic LAN setup using a switch
- Static IP configuration
- Connectivity verification with ping
